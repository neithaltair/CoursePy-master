
Antes de la ejecucion, se necesita instalar el archivo requirements.txt con pip3.

Es necesario que exista el archivo llamado hash.txt que contenga linea por linea cada uno
de los hash, pueden ser md5, sha1, sha256. En caso de encontrarse con un id unico
, sera agregado en el excel y se podra identificar.


USO:
El uso es muy simple: python3 hashVirusTotal.py


El resultado sera un archivo JSON y otro archivo Excel.
